Reflection on this Lab

In this lab, I learned how to properly structure and style a table and a multi step form. For the table I learned the importance of accessibility by using thead and tbody and by adding the scope="col" and scope="row" attributes to my headers so screen readers can correctly read the data. For the form, I practiced making it accessible by connecting every label to its input using the for and id attributes.
