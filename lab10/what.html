What is the PNG Standard?

Portable Network Graphics (PNG) is a web standard for a raster graphics file format. It was first made in the 1990s as a replacement for the GIF format. The standard defines a way to store images by compressing them without losing any data. Lossless compression means the quality of the image remains exactly the same every time it is saved or viewed unlike lossy formats like JPEG which discard data.

The standard describes the technical details of the file format like its chunk based structure. A PNG file is made of a signature followed by a series of chunks which store different information. For example the IHDR chunk holds basic information like the image's width, height, and color type. The IDAT chunks store the actual image pixels and other chunks can store metadata like gamma correction (gAMA) or text (tEXt). This structure is important because it makes the format extensible which allows new features to be added later.

One of the most well known features defined by the PNG standard is its support for an alpha channel. The alpha channel allows each pixel to have a level of transparency from completely opaque to completely transparent. This is a massive advantage for web design since it allows images to blend smoothly over different website backgrounds. The new Third Edition of the standard also introduces formal support for high-dynamic-range (HDR) images and suggests ways to handle simple animations which keeps the format modern.
