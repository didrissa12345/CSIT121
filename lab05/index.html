Lab 5 Reflection: Best Design Practices

In Lab 5 I focused on making my website responsive. I learned how to use a media query in my CSS file to change the layout based on the screen size. For mobile I kept a simple single-column layout. For desktops, the media query activated Flexbox which rearranged my content into a multi column layout. I also learned how to create a navigation menu using anchor links to let people jump to different sections of the same page.
